# Laboratory System (Учет химических реактивов)

Консольное приложение (CLI) на Java для управления запасами химической лаборатории. Позволяет вести справочник реагентов, учитывать конкретные партии (банки/флаконы), отслеживать их местоположение, сроки годности и историю расхода.

## Основной функционал

* **Справочник реагентов:** Добавление химических веществ (Название, Формула, CAS, Класс опасности) и поиск по ним.
* **Учет партий (Batches):** Регистрация новых поступлений, указание места хранения, отслеживание остатка (в граммах или миллилитрах) и сроков годности.
* **Движение остатков:** Оформление прихода (`IN`), расхода (`OUT`) и списания (`DISCARD`) с указанием причины.
* **Архивация:** Перемещение пустых или просроченных партий в архив без удаления из истории.
* **Отчеты:** Генерация сводки по складу с возможностью фильтрации по сроку годности.

---

## Технологии и архитектура

* **Язык:** Java 17+
* **Сборка:** Gradle (Kotlin DSL)
* **Тестирование:** JUnit 5 (зависимости подключены)
* **Хранение данных:** In-Memory (данные хранятся в оперативной памяти с помощью `LinkedHashMap` и сбрасываются при закрытии программы).
* **Архитектура:** Layered Architecture (Domain, Repository, Service, Validation, CLI).

## Установка и запуск

Поскольку проект использует хранение данных в памяти, для его запуска не нужны базы данных или сложные настройки.

### Вариант 1: Запуск через IDE (IntelliJ IDEA, Eclipse, VS Code)
1. Склонируйте репозиторий.
2. Откройте проект в вашей IDE как Gradle-проект.
3. Дождитесь загрузки зависимостей.
4. Запустите класс `ru.Meowteam.lab.Main`.

### Вариант 2: Запуск через терминал (с помощью Gradle)
